*Project Title:
Project Management System in C 
 *Functional Requirements:
1. Add Task

Enter Task ID

Enter task priority (1–5)

Enter deadline (day, month, year)

Status defaults to Pending (0)

Progress defaults to 0%

2. View All Tasks

The system displays:

Task ID

Priority

Deadline (dd-mm-yyyy)

Task Progress (0–100%)

Status (Pending / Completed)
3. Update Task Status

User enters Task ID

User sets status → 0 (Pending) or 1 (Completed)

4. Update Task Progress

User enters Task ID

User sets progress → 0 to 100

5. Exit the Program
*Features of the Project:
1. No String Usage

Entire program works using integers only

Avoids char arrays, string.h, or text input
2. Uses Structure for Storing Task Data

*Struct stores:

ID

Priority

Status

Progress

Deadline (dd/mm/yyyy)
 3. Menu-Driven Interface

Easy to navigate

User chooses from 5 simple options
 4. Scalable (100 Tasks Supported)

Uses an array of 100 structures

Can be increased if needed
 5. Fully Console-Based

Works on any C compiler

No external library required.























End the application safely.
